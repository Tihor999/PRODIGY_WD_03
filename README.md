# Tic Tac Toe Game

Welcome to the Tic Tac Toe game project! This is a simple command-line implementation of the classic Tic Tac Toe game, developed using Python. The game allows two players to take turns marking the spaces in a 3Ã—3 grid with X or O. The player who succeeds in placing three of their marks in a horizontal, vertical, or diagonal row wins the game.

## Features

- Two-player mode: Play against another human.
- Simple and intuitive command-line interface.
- Input validation to ensure valid moves.
- Automatic detection of win and draw conditions.
